# 温馨家庭网站

一个简洁温馨的家庭记忆网站，让家庭成员可以上传图片并记录美好的家庭回忆。

## 功能特点

### 首页
- 温馨的视觉设计，采用暖色调配色
- "敬请期待"的占位页面
- 功能预览卡片

### 回忆功能
- 按年份、月份组织回忆
- 支持上传本地图片
- 添加文字描述
- 按日期倒序展示
- 点击查看回忆详情
- 支持删除回忆

## 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架（通过 CDN 引入）
- **JavaScript** - 交互逻辑
- **LocalStorage** - 数据存储

## 使用方法

### 直接打开

1. 在浏览器中打开 `index.html` 文件即可使用

### 使用本地服务器（推荐）

如果你想使用图片上传功能，建议使用本地服务器：

```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 或使用 Node.js (需要先安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 设计说明

### 配色方案
- **主色调**: 温暖的琥珀色 (#F59E0B, #FCD34D)
- **背景色**: 柔和的浅黄色渐变 (#FEF3C7 → #FDE68A)
- **文字色**: 深灰色 (#1F2937) 保证可读性

### 字体
- **中文标题**: Noto Serif SC（思源宋体）- 优雅温馨
- **中文正文**: Noto Sans SC（思源黑体）- 清晰易读

### 交互设计
- 平滑的过渡动画
- 卡片悬停效果
- 响应式布局，适配移动端和桌面端
- 模态框展示回忆详情

## 数据存储

所有回忆数据存储在浏览器的 LocalStorage 中，因此：
- 数据保存在本地浏览器中
- 清除浏览器数据会丢失回忆
- 可以导出数据进行备份
